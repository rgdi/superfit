# Manual de usuario — SuperFit

## ¿Qué es SuperFit?

Una app 100% offline que te dice la rutina del día en el gimnasio y se adapta a tu rendimiento.

## Flujo básico

1. **Abre la app** → ves la rutina del día
2. **Pulsa EMPEZAR** → arranca la sesión con cronómetro
3. **Para cada ejercicio**:
   - Ves la imagen con los músculos activados
   - Registras peso, reps y RPE post-set
   - El timer de descanso arranca solo
4. **Termina la sesión** → la app guarda todo y actualiza tu plan
5. **Vuelve al día siguiente** → la app sugiere qué toca

## Conceptos clave

### RPE (Rating of Perceived Exertion)
- **7** = podrías hacer 3 reps más (mínimo efectivo)
- **8** = podrías hacer 2 más (óptimo hipertrofia)
- **9** = podrías hacer 1 más (alto)
- **10** = fallo absoluto

Apunta RPE honestamente después de cada set.

### Doble progresión
- Si completas todas las reps a RPE ≤8 dos veces seguidas, **sube 2.5 kg**
- Si RPE ≥9, **mantén o baja**
- Si 3 semanas estancado, la app propone cambiar de ejercicio

### Deload
- Cada 4 semanas: reduce 50% volumen, RPE 6-7
- Semana de recuperación, sigue entrenando

## Galería de progreso

- Toma fotos cada 2-4 semanas
- Compara lado a lado
- Registra peso, perímetros, % grasa

## Privacidad

- **0 cuentas** · **0 nube** · **0 tracking**
- Tus datos viven en tu dispositivo
- Puedes exportar todo como JSON (en settings)
- Puedes borrar todo con un click

## Soporte

Esta es una app open-source sin soporte oficial. Si encuentras bugs, repórtalos en el repo.
